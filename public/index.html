언제 알림을 보낼까요?

백그라운드 감시 안내 및 절전 방지 팁

  • 화면을 닫아도 감시 유지: 감시 엔진은 관리 서버(백엔드)에서 24시간 작동합니다. 스마트폰 화면이나 브라우저 창을 완전히 닫아도 백그라운드에서 감시가 수행되며, 구매 가능 상태가 되면 알림을 전송합니다.
  • 실시간 푸시 알림 수신: 알림을 받으시려면 반드시 우측 '브라우저 알림' 패널에서 [알림 켜기]를 눌러 기기를 등록하고 기기 자체의 알림 권한을 승인해 주셔야 합니다.
  • 무료 호스팅(Render 등) 절전 해결: 무료 클라우드 플랫폼을 사용할 경우 약 15분 동안 외부 접속이 없으면 서버가 자동으로 대기 상태(Sleep)로 전환되어 감시가 멈춥니다. 24시간 연속 감시를 하려면 UptimeRobot 같은 무료 모니터링 사이트에 가입하여 이 서버의 헬스체크 API 주소(/api/health)를 5분 주기로 찌르도록 등록해 주세요.
브라우저 알림은 기기 등록 후 백그라운드로 전달됩니다.
